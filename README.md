# Misty Greens – Nature’s Luxury Escape

A Next.js (App Router) + TypeScript + Tailwind + shadcn-style UI website for Misty Greens (Sri Lanka).

## Quick start

```bash
npm install
npm run dev
```

## Configure WhatsApp + Email

Create `.env.local`:

```bash
NEXT_PUBLIC_WHATSAPP_NUMBER=9477XXXXXXX
NEXT_PUBLIC_EMAIL=your@email.com
```

WhatsApp booking buttons open a pre-filled message to your number.

## Pages

- `/` Home
- `/properties` Properties list
- `/properties/[slug]` Property details
- `/contact` Contact
